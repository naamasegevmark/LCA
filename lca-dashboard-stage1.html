Scope Completeness Checklist
Confirm each element of scope definition per ISO 14044. Check off each item as you have addressed it.
  • Product system definedThe material / construction system to be studied is clearly described.
  • Function of the system statedMain and secondary functions are identified.
  • Functional unit is quantifiedPerformance, duration, and scale are all specified.
  • System boundary covers same functional reality in all scenarios(ISO Rule 1)
  • Cut-off criteria definedProcesses contributing <x% are excluded with justification.
  • Allocation procedures statedMethodology chosen and documented.
  • LCIA methodology selectede.g. ReCiPe, CML, TRACI (for Stage 3)
  • Data quality requirements definedAge, geography, technology representativeness specified.
  • Peer review type decidedInternal / panel / independent expert.
